Probably the most complete *crosstab* solution for Vue.js 2.0, without jQuery.

## Features & characteristics:
* Fixed headers
* Portial rendering
* Custom themes

## Documentation and Examples
http://amalieiev.github.io/vue-crosstab

## Install & basic usage

```bash
npm install vue-crosstab
```
